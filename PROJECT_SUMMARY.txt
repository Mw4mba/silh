🔧 CUSTOMIZATION QUICK TIPS
═════════════════════════════════════════════════════════════════════════════

Change Colors:
  → Edit :root variables in globals.css
  → Update Tailwind className hex values

Update Content:
  → Edit text directly in component files
  → Modify services array in ServicesSection.tsx
  → Modify projects array in ProjectsSection.tsx

Add Sections:
  → Create new component in components/
  → Import and add to page.tsx
  → Use the same animation patterns

Adjust Navbar Width:
  → Find "lg:w-64" in Navbar.tsx
  → Change to desired width (e.g., "lg:w-80")

Modify Animations:
  → Edit duration in globals.css keyframes
  → Adjust GSAP settings in component files
  → Change ease functions as needed
